# 감정 탐정 (Emotion Detective)

몇 개의 짧은 질문으로 지금 감정에 가장 가까운 **후보들**을 찾아주는 한국어 감정 탐색 도구입니다. 하나로 단정하지 않고, 후보를 확률과 확신도로 제시합니다.

**Live:** https://woonho0203.github.io/emotion-detective/

## 특징
- **규칙기반 결정트리 + 후보 확률분포** — 정보이득 기반 동적 분기, 매 응답마다 후보 확률(softmax) 갱신
- **탐정 은유 + 따뜻한 상담 톤** — 한 화면 한 질문, 낮은 기록 피로
- **안전 최우선** — 위기 신호 시 일반 흐름을 멈추고 상담 연결(109 · 1577-0199 · 1388 · 119) 우선
- **결과 PDF 저장** — 폰에서도 인쇄→PDF로 저장
- **연구·검증용 응답 로그** — 브라우저 로컬 저장, JSON/CSV 내보내기(동의 세션만), top-1/top-3 자기확인

## 성능 (16종 시뮬레이션)
top-1 15/16 · top-3 16/16 · 평균 8.6문

## 구현
백엔드·API 없는 자체완결 정적 사이트(단일 `index.html`). 모든 감정 데이터·문항·엔진 파라미터는 헌법형 지침체계의 고시에서 도출.

## 고지
이 서비스는 감정 이해와 자기기록을 돕는 **정보제공 도구**이며, 의료적 진단이나 치료를 대신하지 않습니다. 결과는 확정된 감정이 아니라 참고용 후보입니다. **연구·검증용**이며, 불특정 다수 대상 공개·상업 배포 전에는 의료·개인정보 전문 검토가 필요합니다.
